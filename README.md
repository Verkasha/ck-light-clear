# ЦК Простой шаблон библиотеки для работы с базами данных

## Требования
- Python 3.10+

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```python
python main.py
```

## Отредактируйте файлы
- database/migrations/create_book_table.py
- app/routes/book.py
